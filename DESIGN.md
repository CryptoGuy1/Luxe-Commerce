## âœï¸ Typography

### Font Families

**Display Font:** Playfair Display (Serif)
- Used for headings, hero text, and brand elements
- Conveys elegance and sophistication

**Body Font:** DM Sans (Sans-serif)
- Used for body text, navigation, and UI elements
- Clean, modern, highly readable

**Monospace:** JetBrains Mono
- Used for prices, codes, and technical details

### Type Scale

| Style | Size | Font |
|-------|------|------|
| Display XL | 5rem (80px) | Playfair Display |
| Display LG | 3.75rem (60px) | Playfair Display |

| Body LG | 1.125rem (18px) | DM Sans |
| Body | 1rem (16px) | DM Sans |
| Body SM | 0.875rem (14px) | DM Sans |
